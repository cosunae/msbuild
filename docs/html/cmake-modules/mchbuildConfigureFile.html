msbuild_configure_file¶

Configure a file and store the output in the same directory hirarchy as the input file while substituting CMAKE_SOURCE_DIR for CMAKE_BINARY_DIR. If the input file has a .cmake extension, it will be removed in the output. This will also handle the installation of the file correctly by replacing src/ with include/.

msbuild_configure_file(FILE)
FILE
File to configure.

Example¶

msbuild_configure_file(${CMAKE_SOURCE_DIR}/src/msbuild/Config.h.cmake)

Ths stores the configured file in ${CMAKE_BINARY_DIR}/src/msbuild/Config.h and, in addition, it will install it in ${CMAKE_INSTALL_PREFIX}/include/msbuild/Config.h” (hence it will strip "src/").
